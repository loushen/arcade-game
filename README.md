#### 安装教程

#### 使用说明

1.  不限速度下载，不用像百度云开会员；不需要注册账号，下载即可
2.  没有长期存在的网站，链接容易失效，所以自己上传文件，只要本人账号还在就不会失效
3.  由于100M上传限制，上传的文件不能超过100M，所以本人删除了几个里面超过100M的游戏；望谅解
